# RuStore - Российский магазин приложений для Android

Мобильное приложение- MVP версия RuStore с использованием современных Android технологий.

## Технологии

- **Язык**: Kotlin
- **UI**: Jetpack Compose
- **Архитектура**: MVVM (Model-View-ViewModel)
- **Навигация**: Jetpack Navigation Component
- **Асинхронность**: Kotlin Coroutines, Flow
- **DI**: Внедрение зависимостей через ViewModel
- **Хранение данных**: SharedPreferences, Local Data

## Запуск проекта

### Требования
- Android Studio Arctic Fox или выше
- Android SDK 31+
- Kotlin 1.8+

### Шаги
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/hopemask0/RuStore

2. Откройте проект в Android Studio
3. Запустите проект на эмуляторе
